# Intelliwaste Event Planner

An AI-powered sustainability dashboard that estimates event food waste and provides actionable reduction strategies using IBM Watsonx AI.

## Features
- **Dynamic Waste Estimation**: Parses menu inputs to estimate waste based on menu type.
- **AI-Powered Insights**: Uses Llama-3-3-70b to generate sustainability recommendations.
- **Real-time Visualization**: Interactive dashboard to track Carbon Footprint and Recovery Potential.

## Tech Stack
- **Frontend**: Streamlit
- **Visualization**: Plotly
- **AI Engine**: IBM Watsonx AI
